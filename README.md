## 📱💻 Instruções

```
## 1. Clonar repositório
git clone https://github.com/guivictorr/weather-app.git

## 2. Entrar na pasta
cd weather-app

## 3. Instalar as dependências
yarn

## 4. Rodar a aplicação
yarn start
```
